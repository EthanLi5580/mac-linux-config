Installation:
------------
    git clone git://github.com/yuchih/dotfiles.git ~/.vim

Create symlinks:
---------------

    ln -s ~/.vim/vimrc ~/.vimrc
    ln -s ~/.vim/LS_COLORS ~/.LS_COLORS
    ln -s ~/.vim/gitconfig ~/.gitconfig
    ln -s ~/.vim/bash_profile ~/.bash_profile
